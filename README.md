## 🌟 Overview

**LensGallery** is an avant-garde portfolio template designed to replicate a high-end physical art gallery in the digital space. Originally conceived via GSAP and vanilla HTML, it has been painstakingly engineered into a highly scalable, modern **React Architecture**.

Every scroll, hover, and interaction is intentionally designed to place the focus entirely on the artwork, utilizing physics-based animations, customized masking, and absolute performance via **Vite**.

## ✨ Core Aesthetic Features

- 🎭 **Immersive Atmosphere:** A sleek dark-mode aesthetic layered with an authentic SVG film-grain overlay (`mix-blend-mode`).
- 🖱️ **Custom Kinetic Cursor:** A highly responsive, custom-built cursor that morphs and utilizes `mix-blend-mode: difference` on interactive elements.
- 🎞️ **Horizontal Scroll Series:** A spectacular GSAP/Framer-inspired horizontal scrolling exhibit, allowing users to linearly explore curated collections.
- 🧱 **Masonry Exhibit Grid:** A fully responsive masonry layout that elegantly displays works in varied aspect ratios, complete with parallax hover reveals.
- 🔦 **Cinematic Lightbox:** An edge-to-edge interactive modal for undisrupted, full-screen contemplation of individual images.
- 🖋️ **Premium Typography:** Utilizing a meticulous pairing of _Playfair Display_ (for dramatic editorial headings) and _Plus Jakarta Sans_ (for pristine micro-copy UI).

## 🚀 Getting Started (Local Development)

To run this elite project locally and begin customizing your own digital gallery, follow these steps:

### 1. Clone the Source

```bash
git clone https://github.com/Sameerkaradbhajne/image-gallary.git
cd image-gallary
```

### 2. Install Project Dependencies

Ensure you have `Node.js` installed. Then run:

```bash
npm install
```

### 3. Ignite the Dev Server

```bash
npm run dev
```

> Navigate to `http://localhost:8080` (or your terminal-specified port) to experience the live build.

---

## 🏗️ Building for Production

When you are ready to deploy your masterpiece to the world (e.g., via Vercel, Netlify, or AWS), generate an optimized static bundle:

```bash
npm run build
```

_The resulting assets will be output securely to the `/dist` directory, fully minified and prepped for CDN delivery._
